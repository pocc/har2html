Tools > HAR To HTML

  • To Use: Choose a HAR file to convert to HTML.
  • How: This page takes the text payload of responses and adds them to a single HTML file. This HTML file is then inserted it into an iframe and can be downloaded.
  • Why: I troubleshoot technical issues at Cloudflare and work with HAR files frequently. A loaded HTML page can visually provide more context to an issue than just the HAR file.
